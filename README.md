📌 Overview

This project demonstrates a complete end-to-end data analytics workflow — from raw data processing to business intelligence reporting.

The objective of this project is to:

Load and preprocess raw data using Python

Perform Exploratory Data Analysis (EDA)

Clean and transform the dataset

Execute SQL queries using PostgreSQL

Build an interactive Power BI dashboard

Create a detailed analytical report

Develop a professional presentation using Gamma

This project highlights skills in data analysis, SQL, data visualization, and business reporting, making it recruiter-friendly and industry-relevant.

🗂️ Dataset

Structured dataset (CSV format)

Contains business-related transactional/operational data

Includes categorical and numerical features

Used for extracting insights, trends, and performance metrics

(Dataset can be replaced with any business dataset such as sales, hotel booking, vendor performance, etc.)

Filtering business KPIs

Trend analysis

Extracted meaningful insights for reporting
